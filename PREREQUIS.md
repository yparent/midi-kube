# Avant de commencer — les bases qui aident

Pas besoin d'être expert. Juste être à l'aise avec ça. Si un point est flou,
10 min de révision et tu suis tout le reste sans souci.

## Terminal / Linux
Ouvrir un terminal et taper une commande. Les 5 utiles pour la série :
- `cd` — se déplacer dans les dossiers
- `ls` — lister ce qu'il y a
- `cat` — afficher le contenu d'un fichier
- `whoami` — savoir qui je suis (on s'en sert beaucoup à l'ép. 1 !)
- `clear` — nettoyer l'écran

## git
Une seule commande pour suivre la série :
```bash
git clone https://github.com/yparent/midi-kube.git
```

## Réseau
Juste deux notions pour l'instant :
- un **port** (le `8080` dans nos exemples) = la « porte » par laquelle on parle à l'app
- **localhost** = ta propre machine

## YAML
La seule chose à retenir maintenant :
- l'**indentation** donne le sens (ce qui est décalé à droite « appartient » à la ligne du dessus)
- on indente avec des **espaces**, **jamais** de tabulations (l'erreur n°1 de tout le monde)
- un tiret `-` = un élément de liste

C'est tout. On explique le reste en live, fichier sous les yeux.
